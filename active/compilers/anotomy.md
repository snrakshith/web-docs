## Modern Approach for Compiler Anotomy is in 3 Phases:

- Frontend
  - Lexer ( Tokenizer )
  - Parser ( Syntax )
  - Symantic
    - Type Checking
- Middle
  - Intermediate Representation
  - Optimizer
    - Platform Independent
- Backend
  - Optimizer
    - Platform Dependent
  - Code Analysis
  - Code Generator
