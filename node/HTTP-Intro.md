# Status Codes

- 1.xx Informational
- 2.xx Success
  - 200 Success
  - 201 Created
  - 204 No Content
- 3.xx Redirectional
  - 304 Not Modified
- 4.xx Client Error
  - 400 Bad Request
  - 401 Unauthorized
  - 404 Not Found
- 5.xx Server Error
  - 500 Internal Server Error
